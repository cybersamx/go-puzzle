# Binary Gap

This a Codility exercise 1-1.

Given an integer n, convert it to binary, and then return the count of the longest sequence of zeros.

## Output

| Input (Binary) | Output (Zero Count) |
|----------------|---------------------|
| 0              | 1                   |
| 1111           | 0                   |
| 10001          | 3                   |
| 10100          | 2                   |
| 10010          | 2                   |
